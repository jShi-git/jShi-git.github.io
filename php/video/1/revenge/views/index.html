联手对抗臭长广告

这是什么？

一句话：提取各大视频网站的真实视频地址。

为什么要做这个？

最初，这个东西是实习的时候给公司做的视频聚合服务（央视有报道喔 :p 稍后请看最下面「与此相关」），后来发现提取的视频地址还能机智的跳过广告，（太棒了！）又有一天，惊奇的发现某些网站的会员视频 居然可以直接提取出来播放（Oh,My God,这是一箭三雕么？那个网站的程序员同学，这是你留的后门么？）

为什么要开源？

  • 真心厌恶了那些视频网站又臭又长的广告了。（我摔！）
  • 有时候花五块钱看付费电影，截图秀色可餐，但是看完后却大呼被坑！
  • Mac观看flash视频热的要死，国内的视频网站根本懒得开发html5版本！（还是YouTube强！）
  • 偶尔想下载视频，却发现整个过程好蛋疼，还得去找Pc机器装客户端！（我摔！）
  • 开发视频类App自己建视频服务实在是扛不住流量费用！

可以用来

  • 跳过广告
  • App内播放视频
  • 看付费
  • 给Mac降温
  • 轻松下载

使用方法

  1. 获取视频网页地址，如优酷万万没想到第一集：
    http://v.youku.com/v_show/id_XNTkyMTM0MzY4.html
  2. 构造请求串
    http://revenge.daimajia.com/youku/?url=http://v.youku.com/v_show/id_XNTkyMTM0MzY4.html
  3. 请求真实地址

部署方法

  1. 安装Node
  2. git clone git@github.com:daimajia/revenge.git revenge
  3. cd revenge
  4. npm install
  5. node index.js
  6. 访问 http://localhost:3000
